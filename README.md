# CloudLast placeholder

Sito promozionale minimal basato su React e React Router che raccoglie le principali policy e la
proposta di valore CloudLast (privacy, sicurezza, roadmap, changelog e miglioramenti operativi).

## Sviluppo locale

1. Installa le dipendenze (se non presenti):

   ```bash
   npm install
   ```

2. Avvia il dev server Vite (se configurato nel tuo ambiente):

   ```bash
   npm run dev
   ```

Le pagine sono definite in `src/pages/` e condividono lo stile in `src/assets/style.css`.
